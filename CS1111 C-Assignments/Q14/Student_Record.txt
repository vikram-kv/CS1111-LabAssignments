Program's Goal:
To read the data of a set of students from a file , create a linked list dynamically and print out the contents on terminal

Instructions for compilation:-
1.Compile using the GCC compiler with std=c11 and run the .out file generated
2.Input is taken from data.txt which must be in the same directory as the .out file
3.Fields of the same student are delimited by a horizontal tab and different students' records are delimited by a newline.

Sample Input and Output:-
